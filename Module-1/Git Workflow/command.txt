Git & SSH Commands (Cheat Sheet)
SSH Setup
ssh-keygen -t ed25519 -C "your_email@example.com"   # Generate SSH key


Add the key to the agent:

eval "$(ssh-agent -s)"    # Start ssh-agent
ssh-add ~/.ssh/id_ed25519 # Add the private key
ssh-add -l                # Verify keys loaded


Show your public key:

cat ~/.ssh/id_ed25519.pub


Test connection with GitHub:

ssh -T git@github.com

Basic Linux Commands
ls -al     # Show all files (including hidden)
code .     # Open current folder in VS Code

Git Basics
git clone <url>                  # Clone a repository
git status                       # Check changes
git add .                        # Add all changes
git commit -m "message"          # Commit changes
git log                          # Full commit history
git log --oneline                # Short commit history

Branching
git branch                       # Show branches (current highlighted)
git branch -a                    # Show all branches (local + remote)
git checkout -b feature_branch   # Create & switch to new branch
git checkout branch_name         # Switch to existing branch

Push & Reset
git push origin feature_branch   # Push branch to remote
git reset --hard <commit_id>     # Reset to specific commit

Remotes
git remote -v    # Show remotes
git remote --help # Remote help

Syncing & Merging
git fetch                   # Fetch updates
git merge branch_name       # Merge branch into current
git rebase branch_name      # Reapply commits on top of another branch


If conflict occurs:

nano conflict_file_name     # Open conflict file and resolve

Interactive Rebase
git rebase -i HEAD~2        # Edit last 2 commits
